toString()  method
----------------------
1) What  does  toString()  method  do ?  --->
							Concatenates  all  the  values  of  object  to  form  a  string  and   returns  the  string

2) How  is  toString()  method  called ?  --->  object . toString()

3) When  is  toString()  method  automatically  executed  ?  --->
										When  object  is  printed  with  println()  (or)  print()  method

4) System . out . println(object);
    What  is  printed ?  --->  That  string  returned  by  toString()  method

5) System . out . println(object);
    How  is  the  statement  reduced  to ?  --->  System . out . println(string)

6) String  s = "Hyd";
    System . out . println(s);
    Which  class  method  is  executed  ?  --->  toString()  method  of  String  class  becoz  's'  is  String  class  object
    What  does  toString()  method  of  String  class  return ?  --->The  string  in  String  object  i.e.  "Hyd"

7) String  s = "Hyd";
    System . out . println(s);
    How  is  the  statement  reduced  to ?  --->  System . out . println("Hyd");

8) Integer  i = new  Integer(25);
    System . out . println(i);
    Which  class  method  is  executed  ?  --->  toString()  method  of  Integer  class  becoz  'i'  is  Integer  class  object
    What  does  toString()  method  of  Integer  class  return ?  --->
									The  integer  value  in  Integer  object  in  the  form  of  string  i.e. "25"

9) Integer  i = new  Integer(25);
    System . out . println(i);
    How  is  the  statement  reduced  to ?  ---> System . out . println("25");

10) Date  d = new  Date();
     System . out . println(d);
     Which  class  method  is  executed  ?  --->  toString()  method  of  Date  class  becoz  'd'  is  Date  class  object
     What  does  toString()  method  of  Date  class  return ?  --->
								The  current  date , time  and  day  of  the  week  in  the  form  of  string

11) Date  d = new  Date();
      System . out . println(d);
      How  is  the  statement  reduced  to ?  ---> System . out . println("current  date , time  and  day  of  the  week");

12) System . out . println(object . toString());
      System . out . println(object);
      What  is  the  difference  between  the  two  statements ?  --->
					System . out . println(object . toString())  --->  Calls  toString()  method  explicitly
					System . out . println(object)  --->  Automatically  executes  toString()  method
												i.e.  Executes  toString()  method  implicitly

13) String . toString();
     Is  the  statement  valid ?  ---> No  becoz  toString()  is  not  a  static  method  of  String  class

14) System . out . println(variable);
      Is  toString()  method()  executed ?  --->  No  becoz  variable  is  being  printed  but  not  object
