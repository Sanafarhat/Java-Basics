binarySearch()  method
----------------------------
1) What  does  binarySearch(array , valid-element)  do ?  --->  Returns  index  of  the  element  in  the  array

2) What  does  binarySearch(array , invalid-element)  do ?  --->  Returns  -(Index  of  the  element  in  the  array + 1)

3) What  is  the  pre-requisite  to  call  binarySearch()  method ?  --->  Array  has  to  be  sorted

4) Arrays . binarySeacrh(array , x);
    Why  is  binarySearch()  method  called  thru  class  name  Arrays ?  --->  Since  it  is  a  static  method  of  Arrays  class
